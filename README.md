# ServerClient
类似web服务器功能
# 操作
运行代码，在弹出的窗体中点击“开启服务器”
![](https://github.com/HelloTR/WebServer/blob/master/screenshot/1.jpg)
#
打开浏览器，输入http://localhost:8888/resources/helloworld.txt
此时会读取服务器下的helloworld.txt文件
